## Getting Started

## Installation

1. Clone the repo
   ```shell
   git clone git@github.com:Kozakadam/stack-overflow.git
   ```

2. Install the frontend packages
     ```shell
     npm install
     ```

3. Start the application
     ```shell
     npm start
     ```
